This Repository includes the projects assigned during dashLab inductions for the system subsection.
The first part of this repository includes CUDA files that aim to approach cuBLAS level of GEMM optimization, starting from the naive approach all the way to including techniques such as Tiling, Block Registering and Double Buffering.
A progress report is also included detailing the journey from a naive approach to a more optimised one.
The second part of this repository includes Report analysing one the Research Papers provided.
